# Subdirectories

* [datasets](/datasets/) - A sample of obfuscated Google Analytics BigQuery event export data for three months from the Google Merchandise Store is available as a public dataset in BigQuery.
* [references](/references/) - This directory contains specifications for data joins and definitions for user activity and purchase metrics.
* [tables](/tables/) - Contains Google Analytics event export data from the `ga4_obfuscated_sample_ecommerce` dataset.
