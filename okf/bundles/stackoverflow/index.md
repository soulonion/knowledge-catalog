# Subdirectories

* [datasets](/datasets/) - The Stack Overflow public dataset contains a variety of tables related to Stack Overflow user activity, posts, and tags. This dataset is no longer actively updated.
* [references](/references/) - This directory contains enumerated types, data structures, and internal references for Stack Exchange content, moderation, and the Data Explorer.
* [tables](/tables/) - The directory contains structured data tables describing Stack Overflow content, users, and activity.
