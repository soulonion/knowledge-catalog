# Post Type IDs

This document defines the enumerated IDs for different types of posts found in the Stack Exchange data dump and SEDE.

- `1`: Question
- `2`: Answer
- `3`: Orphaned Tag Wiki
- `4`: Tag Wiki
- `5`: Tag Wiki Excerpt
- `6`: Moderator Nomination
- `7`: Privilege Wiki
- `8`: Status Wiki
- `10`: Deleted Question (not in public data dump)
- `11`: Section
- `12`: Article
