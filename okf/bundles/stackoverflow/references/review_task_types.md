This document defines the enumerated types for various review tasks available on Stack Exchange sites.

- `1`: Suggested Edit
- `2`: Close Votes
- `3`: Low Quality Posts
- `4`: First Post
- `5`: Late Answer
- `6`: Reopen Vote
- `7`: Community Evaluation
- `8`: Link Validation
- `9`: Flagged Posts
- `10`: Triage
- `11`: Helper
- `12`: First Questions
- `13`: First Answers
